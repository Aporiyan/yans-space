# 效率工具推荐

分享一些提高工作效率的工具和方法，让日常工作更加高效。

## 代码编辑器

### Visual Studio Code
- 丰富的插件生态
- 强大的调试功能
- 内置终端支持

推荐插件：
- Prettier - 代码格式化
- ESLint - 代码检查
- GitLens - Git 增强
- Live Server - 本地服务器

## 终端工具

### Oh My Zsh
让终端变得更加强大和美观：

```bash
# 安装 Oh My Zsh
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

### 常用命令别名
```bash
alias ll='ls -la'
alias gs='git status'
alias gc='git commit'
alias gp='git push'
```

## 在线工具

### 设计相关
- Figma - 界面设计

- CodePen - 在线代码编辑

## 时间管理

### 番茄工作法
- 25分钟专注工作
- 5分钟短休息
- 每4个番茄后长休息15-30分钟

### 任务管理工具
- Todoist - 任务清单
- Notion - 笔记和项目管理
- Trello - 看板式项目管理

## 总结

选择合适的工具能够显著提高工作效率，但记住工具只是手段，关键还是要养成良好的工作习惯。
