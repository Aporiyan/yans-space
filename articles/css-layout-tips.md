# 现代 CSS 布局技巧

## CSS Grid 布局

CSS Grid 是一个强大的二维布局系统，可以同时处理行和列。

### 基本用法

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-gap: 20px;
}
```

### 响应式网格

```css
.responsive-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 1rem;
}
```

## Flexbox 布局

Flexbox 是一维布局方法，非常适合组件内的元素排列。

### 居中对齐

```css
.center {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

### 自适应导航

```css
.nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
```

## 实用技巧

1. **使用 CSS 变量**：提高代码的可维护性
2. **移动优先**：从小屏幕开始设计
3. **语义化 HTML**：选择合适的 HTML 标签

## 总结

现代 CSS 布局技术让我们能够创建更灵活、更响应式的网页设计。掌握 Grid 和 Flexbox 的组合使用是前端开发的重要技能。
